Spoolman API

Stellt eine Spoolman-kompatible REST-API bereit, damit externe Tools (Moonraker, OctoPrint etc.) auf FilaMan zugreifen können.

API-Status

Aktiv

Externe Clients können folgende Basis-URL verwenden:

Endpunkte: /api/v1/vendor, /api/v1/filament, /api/v1/spool, /api/v1/info, /api/v1/health

IP-Zugriffskontrolle

Beschränke den Zugriff auf die Spoolman API auf bestimmte IP-Adressen. Wenn deaktiviert, kann jede IP-Adresse zugreifen.

IP-Filter aktivieren
Nur Anfragen von erlaubten IP-Adressen zulassen

Einzelne IPs, CIDR-Bereiche (z.B. 192.168.1.0/24) oder * für alle.

Keine IP-Adressen konfiguriert
